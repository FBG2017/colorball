# colorball

一款C#开发的桌面小游戏，还是2010年写的，灵感来自于一款手机游戏。

# 运行相关

使用`Visual Studio 2010`开发，最低要求`.Net Framework 4.0`，有生成好的exe文件可供直接运行，最新版本为v1.4。

# 其它说明
